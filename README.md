# WITTEN – GitHub Pages Ready

Diese Version ist für GitHub Pages vorbereitet.

## Wichtig beim Hochladen
Lade den INHALT dieses Ordners ins Repository hoch, nicht den Ordner selbst.

Im Hauptverzeichnis des Repositories müssen liegen:

- index.html
- style.css
- script.js
- .nojekyll
- 404.html
- assets/

## GitHub Pages Einstellungen
Settings → Pages:

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)
- Custom domain: leer lassen
